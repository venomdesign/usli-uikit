Textarea

A multi-line text input that integrates with Angular Reactive Forms.

Default

<usli-textarea placeholder="Enter your message..." />

Custom rows

<usli-textarea [rows]="6" placeholder="Tall textarea..." />

With FormField

<usli-form-field label="Notes">
  <usli-textarea [formControl]="notesCtrl" />
</usli-form-field>

API

NameTypeDefaultDescription
placeholderstring''Placeholder text.
rowsnumber3Number of visible text rows.
errorMessagestring | undefinedundefinedStandalone error message.
